# hobby-site

多言語対応したブログやメディアサイトに使えるテンプレートを作成する

## 使用技術

- Nuxt.js
  - nuxt-content
  - Vue-i18n
- Firebase

### 環境構築

1. `npm i`
2. `npm run dev`


### デプロイ

`firebase login` している状態で

1. `npm run generate`
2. `firebase deploy`
